# OPENSW프로젝트
## 개발 권고사항
1. 각자 fork한 후 구현 완료되면 PR(Pull Request)
2. Python venv를 사용한 패키지 관리(Virtual Environment)
3. Chat GPT 적극 활용
4. Git issue 사용해서 개발 중 발생한 issue 적극적으로 공유
